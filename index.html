Perguntas Frequentes

Com que frequência devo executar uma verificação de vírus?

+

Recomenda-se executar uma verificação completa de vírus pelo menos uma vez por semana. Se você navega frequentemente na internet ou baixa muitos arquivos, considere fazer verificações mais frequentes ou ativar a proteção em tempo real.

Como posso criar senhas fortes e seguras?

+

Comprimento mínimo de 12 caracteres, misturar letras maiúsculas e minísculas; incluir números e símbolos (!, @, #); não usar dados pessoais como nome e data de nascimento; usar frases complexas.

Qual a diferença entre backup completo, backup incremental e backup diferencial?

+

Backup completo: Copia todos os arquivos selecionados. Incremental: Copia apenas arquivos alterados desde o último backup (de qualquer tipo). Diferencial: Copia arquivos alterados desde o último backup completo. O completo oferece a recuperação mais fácil, enquanto os outros economizam espaço.

Qual a diferença entre vírus, worms, trojan, spyware e ransomware?

+

vírus são programas maliciosos que infectam arquivos; worms se espalham por redes sem intervenção do user, trojans são malwares disfarçados de softwares legítimos; spyware coleta dados sem permissão do user e ransonware bloqueia dados e exige um resgate.
